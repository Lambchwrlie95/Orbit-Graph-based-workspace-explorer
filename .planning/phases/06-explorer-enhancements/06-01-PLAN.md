  <files>frontend/src/hooks/useViewPersistence.ts</files>
  <action>Create a custom hook for persisting and retrieving view mode per folder path.

**Implementation**:
- Hook signature: `useViewPersistence(folderPath: string, defaultView: ExplorerViewMode)`
- Returns: `{ viewMode, setViewMode, isLoaded }`
- Persist to localStorage with key pattern: `orbit:view:${folderPath}`
- Handle SSR/localStorage unavailability gracefully
- Debounce saves to avoid excessive writes
- Provide `clearViewPreference(path)` utility function

**Key considerations**:
- localStorage may be unavailable during SSR/hydration
- Handle null/undefined folder paths
- JSON parse/stringify for storage
- Default to provided defaultView if no preference stored

**Code structure**:
```typescript
const STORAGE_KEY = (path: string) => `orbit:view:${path}`;

export function useViewPersistence(folderPath: string, defaultView: ExplorerViewMode) {
  const [viewMode, setViewMode] = useState<ExplorerViewMode>(defaultView);
  const [isLoaded, setIsLoaded] = useState(false);
  
  // Load from localStorage on mount
  useEffect(() => {
    if (!folderPath) return;
    try {
      const stored = localStorage.getItem(STORAGE_KEY(folderPath));
      if (stored) setViewMode(stored as ExplorerViewMode);
    } catch { /* ignore */ }
    setIsLoaded(true);
  }, [folderPath]);
  
  // Save to localStorage when viewMode changes
  useEffect(() => {
    if (!isLoaded || !folderPath) return;
    try {
      localStorage.setItem(STORAGE_KEY(folderPath), viewMode);
    } catch { /* ignore */ }
  }, [viewMode, folderPath, isLoaded]);
  
  return { viewMode, setViewMode, isLoaded };
}
```

   - Default: Name ascending

6. **Parent navigation**:
   - ".." row at top if not at root
   - Same styling as List view parent row

   - Arrow keys navigate grid (up/down/left/right)
   - Enter to open/select
   - Home/End for first/last item
