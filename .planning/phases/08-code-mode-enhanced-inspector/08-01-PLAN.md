In src-tauri/src/commands/file.rs (create if doesn't exist), add:

```rust
use std::fs;
use std::path::Path;

#[tauri::command]
pub async fn read_file_for_edit(path: String) -> Result<String, String> {
    fs::read_to_string(&path)
        .map_err(|e| format!("Failed to read file: {}", e))
}

#[tauri::command]
pub async fn save_file(path: String, content: String) -> Result<(), String> {
    fs::write(&path, content)
        .map_err(|e| format!("Failed to save file: {}", e))
}
```

## Trust Boundaries

| Boundary | Description |
|----------|-------------|
| User input → File save | File content from editor crosses to filesystem write |
| File path → Tauri command | Path validation required before file operations |

## STRIDE Threat Register

| Threat ID | Category | Component | Disposition | Mitigation Plan |
|-----------|----------|-----------|-------------|-----------------|
| T-08-01 | Injection | save_file command | mitigate | Validate file path is within workspace root before writing; reject paths with '..' traversal |
| T-08-02 | Elevation | File read/write | mitigate | Use Tauri's filesystem scope to restrict access to workspace directory only |
| T-08-03 | DoS | Large file handling | accept | Monaco handles large files reasonably; document file size limits |
| T-08-04 | Info Disclosure | File content in memory | accept | Local app only; no network transmission of file content |
</threat_model>
