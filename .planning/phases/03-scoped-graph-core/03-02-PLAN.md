# 03-02: Sigma.js Rendering, Styling, Pan/Zoom, and Node Interactions

## Objective

Render the Graphology graph using Sigma.js with proper styling, pan/zoom controls, and interactive node behaviors including click to inspect and double-click to open.

## Context

Sigma.js renders the Graphology graph in a canvas. Nodes need distinct styling based on type (folder vs file vs cluster) and file extension. Users must be able to click nodes to inspect them and double-click to open files or focus folders.

## Tasks Completed

### Task 1: Sigma Renderer Setup

Updated `frontend/src/components/GraphView.tsx`:

- Initialized Sigma renderer with proper settings
- Enabled camera controls (pan with drag, zoom with wheel)
- Set up label rendering with appropriate colors
- Configured node/edge rendering options

### Task 2: Node Styling

Implemented comprehensive color coding:

**By Type:**
- **Folders**: Green (#4ade80), size 12px
- **Cluster/Summary nodes**: Amber (#f59e0b), size 18px
- **Files**: Various colors by extension, size 7px

**By Extension:**
- Image files (png, jpg, svg, etc.): Light blue (#38bdf8)
- Code files (rs, ts, js, py, go, java, etc.): Purple (#a78bfa)
- Text/Markdown (md, txt): White (#f8fafc)
- Config files (json, yaml, xml, etc.): Amber (#fbbf24)
- Stylesheets (css, html): Pink (#f472b6)
- Documents (pdf, doc): Orange (#fb923c)
- Other files: Gray (#94a3b8)

### Task 3: Pan/Zoom Controls

Added visual zoom controls:
- Zoom in button (+)
- Zoom out button (−)
- Reset view button (⌂)
- Display current zoom level (e.g., "150%")

Controls positioned at bottom-right of graph.

### Task 4: Node Interactions

**Click Node:**
- Calls `onSelectPath` callback with node path
- Updates inspector via parent component
- Handles cluster nodes by extracting parent folder path

**Double-Click Node:**
- **Files**: Calls `onOpenPath` to open externally
- **Folders**: Calls `onFocusFolder` to navigate into them
- **Cluster nodes**: Navigate to parent folder

**Hover Node:**
- Shows tooltip with node name and action hint
- Highlights node (larger, lighter color)
- Tooltip positioned at top-right of graph
