# Plan 02-03: Filename Search and Results

## Objective

Implement filename search using SQLite and create flows for selecting and opening search results.

## Current State

- Search input exists in top bar
- `search_files` Tauri command exists (LIKE query)
- Search results displayed in explorer view
- No dedicated search UI

## Implementation Tasks

### Task 1: Create SearchPanel Component

Create a dedicated search interface.

**Files to create:**
- `frontend/src/components/SearchPanel.tsx`

**Features:**
- Search input with debouncing
- Results list with: icon, name, parent folder, size
- Click to select (same as explorer)
- Double-click to open
- Keyboard navigation: arrows to navigate, enter to open
- Clear search button
- Result count indicator
- "No results" state

**Props interface:**
```typescript
interface SearchPanelProps {
  rootPath: string;
  query: string;
  results: FileRecord[];
  selectedPath?: string;
  onQueryChange: (query: string) => void;
  onSelect: (record: FileRecord) => void;
  onOpen: (record: FileRecord) => void;
}
```

### Task 2: Add Search Debouncing

Optimize search to avoid excessive queries.

**Files to modify:**
- `frontend/src/main.tsx`
- `frontend/src/hooks/useDebounce.ts` (create)

**Behavior:**
- Wait 300ms after typing stops before searching
- Cancel in-flight searches on new input
- Show loading state during search

### Task 3: Enhance Backend Search

Improve search functionality in backend.

**Files to verify:**
- `src-tauri/src/db.rs` - `search_files` function

**Enhancements:**
- Case-insensitive search
- Match anywhere in filename
- Limit results (200)
- Order by: folders first, then alphabetically

**Future:** Add FTS5 virtual table for better performance

### Task 4: Wire Search to Mode Switcher

Integrate search into the mode-based UI.

**Files to modify:**
- `frontend/src/main.tsx`

**Behavior:**
- Click Search button in mode bar → focus search input
- Type in global search box → switch to search mode
- Search mode shows SearchPanel instead of explorer
- Keep search query in sync with global search input

### Task 5: Search Result Actions

Enable all actions from search results.

**Files to modify:**
- `frontend/src/components/SearchPanel.tsx`
- `frontend/src/main.tsx`

**Actions:**
- Select: update inspector
- Double-click: open file/folder externally
- Right-click context menu (optional): open, show in folder, copy path

## Verification Steps
